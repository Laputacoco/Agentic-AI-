# 水印去除 Skill

## 概述

本 skill 输出一个**单文件 HTML 工具**，用户下载后在浏览器中双击打开即可：
- 拖放 / 粘贴上传图片
- 在图片上**框选水印区域**（支持多选）
- 一键去除水印并下载结果

所有处理在浏览器本地完成，图片不上传任何服务器。

---

## 核心算法

使用**方向感知外缘取色（Directional Edge Sampling）**：

1. 对选区内每个像素，从上/下/左/右四个方向的选框外缘各取一段颜色样本
2. 对每个方向的样本：计算中位亮度参考色，过滤与参考色差异超过 `outlierThresh` 的异常像素（排除黑色图片边框等干扰）
3. 按距离加权融合四方向颜色：`weight = 1 / dist^dirPow`，越靠近哪条边就越像那边的颜色
4. 边缘软化：选框边缘 `featherR` 像素内做原图与填充色的渐变过渡，消除毛边

**效果**：单色、渐变、深色、复杂背景均能自然还原，不产生色差或毛边。

## 输出规范

生成完整的 HTML 文件，包含以下功能模块：

### 1. 图片上传
- 拖放上传
- 点击选择文件
- Ctrl+V 剪贴板粘贴
- 支持 PNG / JPG / WEBP，最大尺寸缩放至 1600px

### 2. 框选交互
- 鼠标拖动在画布上绘制矩形选区
- 支持触屏（touchstart / touchmove / touchend）
- 多个选区用不同颜色标注编号
- 侧边栏显示选区列表，可单独删除

### 3. 参数控制（4个滑块）

| 参数 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| 外缘取色宽度 | 5px | 2~20 | 向外采样的像素深度 |
| 方向权重 | 4 | 1~8 | 值大=更贴近最近那侧颜色 |
| 边缘软化 | 3px | 0~8 | 消除选框边缘毛边 |
| 异色过滤 | 60 | 0~150 | 过滤异常采样色 |

### 4. 操作按钮
- **去除水印**：执行处理，结果显示在下方
- **下载结果 PNG**：处理完成后可用
- **重新上传**：清空重来

### 5. 结果展示
- 处理结果在页面底部展示
- 状态徽章：待处理 / 处理中… / 完成

```html
      --accent: #d4a96a; --red: #e05252; --green: #52c078; --blue: #5286c0;
      --radius: 10px; --radius-lg: 14px;
    }
    /* 完整样式：body、header、upload-zone、editor grid、
       canvas-panel、sidebar、card、range slider、buttons、
       result-panel、footer、animations */
  </style>
</head>
<body>
  <div class="page">
    <!-- header: logo "WatermarkOut" + "本地处理 · 隐私安全" 标注 -->

    <!-- upload-section: drop-zone(拖放/点击) + paste按钮(Ctrl+V) -->

    <!-- mainArea (上传后显示, display:none初始):
         editor grid (1fr + 260px):
           左: canvas-panel (画布 + 操作提示)
           右: sidebar
               - card: 选区列表 + 清除按钮
               - card: 4个参数滑块
               - card: 去除水印按钮 + 下载按钮 + 重新上传
         result-panel (处理结果展示) -->

    <!-- footer: 工具说明 + 调参建议 -->
  </div>

  <script>
    // ── 1. 图片加载（drag/drop + file input + clipboard paste）
    // ── 2. 框选交互（mousedown/move/up + touchstart/move/end）
    //       redraw(): putImageData → 绘制彩色选区框 + 编号
    //       updateSelUI(): 侧边栏选区列表同步
    // ── 3. 核心算法函数（见下方完整实现）
    // ── 4. runBtn.onclick: 调用 inpaintDirectional → 写入结果canvas
    // ── 5. dlBtn.onclick: canvas.toDataURL('image/png') → a.click()

    function colorDist(r1,g1,b1,r2,g2,b2){
      return Math.sqrt((r1-r2)**2+(g1-g2)**2+(b1-b2)**2);
    }

    // 单方向外缘取色 + 异色过滤
    function sampleEdgeLine(data, W, H, x, y, dx, dy, edgeW, outlierThresh){
      const samples=[];
      for(let d=1;d<=edgeW;d++){
        const nx=x+dx*d, ny=y+dy*d;
        if(nx<0||ny<0||nx>=W||ny>=H) break;
        const ii=(ny*W+nx)*4;
        samples.push([data[ii],data[ii+1],data[ii+2],1/d]);
      }
      if(!samples.length) return null;
      // 中位亮度参考色
      const lums=samples.map(s=>s[0]*0.299+s[1]*0.587+s[2]*0.114);
      const sorted=[...lums].sort((a,b)=>a-b);
      const medLum=sorted[Math.floor(sorted.length/2)];
      let refR=0,refG=0,refB=0,minDiff=Infinity;
      for(const [r,g,b] of samples){
        const d=Math.abs(r*0.299+g*0.587+b*0.114-medLum);
        if(d<minDiff){minDiff=d;refR=r;refG=g;refB=b;}
      }
      // 过滤 + 加权均值
      let rs=0,gs=0,bs=0,ws=0;
      for(const [r,g,b,w] of samples){
        if(outlierThresh>0&&colorDist(r,g,b,refR,refG,refB)>outlierThresh) continue;
        rs+=r*w;gs+=g*w;bs+=b*w;ws+=w;
      }
      return ws>1e-9?[rs/ws,gs/ws,bs/ws]:[refR,refG,refB];
    }

    // 主算法：方向感知外缘取色
    function inpaintDirectional(data, W, H, sels, edgeW, dirPow, featherR, outlierThresh){
      const out=new Uint8ClampedArray(data);
      for(const sel of sels){
        const sx=Math.max(0,sel.x), sy=Math.max(0,sel.y);
        const ex=Math.min(W,sel.x+sel.w), ey=Math.min(H,sel.y+sel.h);
        const sw=ex-sx, sh=ey-sy;
        if(sw<=0||sh<=0) continue;
        // 预采四条边颜色
        const topColor=new Array(sw).fill(null);
        const botColor=new Array(sw).fill(null);
        const leftColor=new Array(sh).fill(null);
        const rightColor=new Array(sh).fill(null);
        for(let c=0;c<sw;c++){
          const px=sx+c;
          topColor[c]=sampleEdgeLine(data,W,H,px,sy,0,-1,edgeW,outlierThresh);
          botColor[c]=sampleEdgeLine(data,W,H,px,ey-1,0,1,edgeW,outlierThresh);
        }
        for(let r=0;r<sh;r++){
          const py=sy+r;
          leftColor[r]=sampleEdgeLine(data,W,H,sx,py,-1,0,edgeW,outlierThresh);
          rightColor[r]=sampleEdgeLine(data,W,H,ex-1,py,1,0,edgeW,outlierThresh);
        }
        // 逐像素插值
        for(let y=sy;y<ey;y++){
          for(let x=sx;x<ex;x++){
            const i=y*W+x, c=x-sx, r=y-sy;
            const dT=Math.max(1,r+1),dB=Math.max(1,sh-r);
            const dL=Math.max(1,c+1),dR=Math.max(1,sw-c);
            const wT=1/Math.pow(dT,dirPow),wB=1/Math.pow(dB,dirPow);
            const wL=1/Math.pow(dL,dirPow),wR=1/Math.pow(dR,dirPow);
            let rs=0,gs=0,bs=0,ws=0;
            const add=(col,w)=>{if(!col)return;rs+=col[0]*w;gs+=col[1]*w;bs+=col[2]*w;ws+=w;};
            add(topColor[c],wT);add(botColor[c],wB);
            add(leftColor[r],wL);add(rightColor[r],wR);
            if(ws<1e-9) continue;
            // 边缘软化
            let alpha=1;
            if(featherR>0){
              const edgeDist=Math.min(dT-1,dB-1,dL-1,dR-1);
              alpha=edgeDist>=featherR?1:Math.max(0,edgeDist/featherR);
            }
            out[i*4]  =Math.round(data[i*4]  *(1-alpha)+(rs/ws)*alpha);
            out[i*4+1]=Math.round(data[i*4+1]*(1-alpha)+(gs/ws)*alpha);
            out[i*4+2]=Math.round(data[i*4+2]*(1-alpha)+(bs/ws)*alpha);
            out[i*4+3]=255;
          }
        }
      }
      return out;
    }
  </script>
</body>
</html>
```

## 参数调整建议

在工具页脚或说明文字中体现以下调参指引：

| 场景 | 建议参数 |
|------|----------|
| 水印在纯色/单色背景 | 外缘宽度 3~4，方向权重 2 |
| 水印在渐变背景 | 方向权重 5~6，边缘软化 3 |
| 水印在深色背景（如黑色） | 异色过滤调小至 30~40 |
| 图片有黑色边框导致毛边 | 异色过滤 40，边缘软化 4 |
| 水印文字较大 | 外缘取色宽度调大至 8~10 |
